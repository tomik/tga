\subsection{Detailed Description}
Manages transposition tables. This class contains transposition tables ( \doxyref{hash\_\-table}{p.}{classHash__table__manager_a7e6ebd5662a7496149125ab6439940d2}) and miscellanous methods / properties to operate these tables :
\begin{DoxyItemize}
\item create and init transposition tables
\item inserting new \doxyref{Hash\_\-table\_\-item}{p.}{classHash__table__item} into tables
\item retrieving existing item ( according to \doxyref{corresponds()}{p.}{t__search_8cc_a015971f4ebc5510a339c2c6011885439} function ) 
\end{DoxyItemize}
