\subsection{Detailed Description}
Class conduncting playing and unplaying moves. This class maintains position presentation, contains data structures neccessary for correct moves (un)playing and provides functions performing different tasks upon position. It also contains several classes defined previously in this header file ( e.g. \doxyref{Ko\_\-manager}{p.}{classKo__manager}, Unplay\_\-node\_\-handle , ... )
