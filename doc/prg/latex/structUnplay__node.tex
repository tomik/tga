\subsection{Detailed Description}
Dynamic structure for unplaying moves. This class creates a dynamic structure of Unplay\_\-manager's where each \doxyref{Unplay\_\-manager}{p.}{structUnplay__manager} represents one move to be unplayed. When there is played new move, new item of \doxyref{Unplay\_\-node}{p.}{structUnplay__node} is created appropriate \doxyref{Unplay\_\-manager}{p.}{structUnplay__manager} is filled with actual information and it is stored in the end of the list.
