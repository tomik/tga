\subsection{Detailed Description}
Handling parametres operations. \doxyref{Params\_\-manager}{p.}{classParams__manager} handles setting appropriate variables according to set parametres on command line. Besides array param\_\-table which stores linking information between parametr and corresponding variable, this class contains all variables set by parametres + functions to operate them -\/ get\_\-.. , set\_\-...
